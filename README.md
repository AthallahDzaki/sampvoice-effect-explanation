# Sampvoice Effect Explanation

# ID
| Nama Fungsi | Definisi | Parameter |
|--|--|--|
| SvEffectCreateChorus | Bekerja dengan menggandakan suara aslinya kemudian menempatkannya sedikit keluar dari waktu dan tune  | SV_INT:priority = Prioritas. <br><br> SV_FLOAT:wetdrymix = Menyetel rasio sinyal basah (diproses) ke sinyal kering (belum diproses).<br><br> SV_FLOAT:depth = Menetapkan persentase waktu tunda yang dimodulasi oleh osilator frekuensi rendah, dalam seperseratus poin persentase.<br><br> SV_FLOAT:feedback = Mengatur persentase sinyal output untuk memberi umpan balik ke input efek.<br><br> SV_FLOAT:frequency = Mengatur frekuensi LFO.<br><br> SV_UINT:waveform = Bentuk gelombang dari LFO.<br><br> SV_FLOAT:delay = Mengatur jumlah milidetik input tertunda sebelum diputar ulang.<br><br> SV_UINT:phase = Mengatur perbedaan fase antara LFO kiri dan kanan|
| SvEffectCreateCompressor | Mengurangi atau menambahkan dynamic range yang berlebihan/kurang | SV_INT:priority = Prioritas. <br><br> SV_FLOAT:gain = Mengatur penguatan output sinyal setelah kompresi.<br><br> SV_FLOAT:attack = Mengatur waktu sebelum kompresi mencapai nilai penuhnya.<br><br> SV_FLOAT:release = Mengatur kecepatan di mana kompresi dihentikan setelah input turun di bawah Ambang Batas.<br><br> SV_FLOAT:threshold = Menetapkan titik di mana kompresi dimulai.<br><br> SV_FLOAT:ratio = Mengatur rasio kompresi.<br><br> SV_FLOAT:predelay = mengatur waktu setelah Ambang tercapai sebelum fase serangan dimulai, dalam milidetik|
| SvEffectCreateDistortion | Distorsi dicapai dengan menambahkan harmonik ke sinyal sedemikian rupa sehingga saat level meningkat, bagian atas bentuk gelombang menjadi kuadrat atau terpotong. | SV_INT:priority = Prioritas. <br><br> SV_FLOAT:gain = Jumlah perubahan sinyal setelah distorsi.<br><br> SV_FLOAT:edge = Persentase intensitas distorsi.<br><br> SV_FLOAT:posteqcenterfrequency = Frekuensi tengah penambahan konten harmonik.<br><br> SV_FLOAT:posteqbandwidth = Lebar pita frekuensi yang menentukan jangkauan penambahan konten harmonik.<br><br> SV_FLOAT:prelowpasscutoff = Filter cutoff untuk redaman harmonik frekuensi tinggi. |
| SvEffectCreateEcho | Efek gema menyebabkan seluruh suara diulang setelah penundaan tetap. | SV_INT:priority = Prioritas. <br><br> SV_FLOAT:wetdrymix =  Rasio sinyal basah (diproses) dengan sinyal kering (tidak diproses).<br><br> SV_FLOAT:feedback = Persentase output yang diumpankan kembali ke input.<br><br> SV_FLOAT:leftdelay = Penundaan untuk saluran kiri, dalam milidetik.<br><br> SV_FLOAT:rightdelay = Penundaan untuk saluran kanan, dalam milidetik.<br><br> SV_BOOL:pandelay = Nilai yang menentukan apakah akan menukar penundaan kiri dan kanan dengan setiap gema yang berurutan. Nilai defaultnya adalah nol, artinya tidak ada swap. |
| SvEffectCreateFlanger | Flange adalah efek gema di mana penundaan antara sinyal asli dan gemanya sangat singkat dan bervariasi dari waktu ke waktu. Hasilnya kadang-kadang disebut sebagai suara menyapu. Istilah flensa berasal dari praktik meraih flensa dari gulungan pita untuk mengubah kecepatan. | SV_INT:priority = Prioritas. <br><br> SV_FLOAT:wetdrymix =  Rasio sinyal basah (diproses) dengan sinyal kering (tidak diproses).<br><br> SV_FLOAT:depth = Persentase waktu tunda yang dimodulasi oleh osilator frekuensi rendah (LFO), dalam seperseratus poin persentase.<br><br> SV_FLOAT:feedback = Persentase sinyal output yang akan di-feed back ke input efek.<br><br> SV_FLOAT:frequency = Frekuensi LFO.<br><br> SV_UINT:waveform = Bentuk gelombang dari LFO.<br><br> SV_FLOAT:delay = Jumlah milidetik input tertunda sebelum diputar ulang.<br><br> SV_UINT:phase = Diferensial fase antara LFO kiri dan kanan. |
| SvEffectCreateGargle | Efek kumur dari modulasi amplitudo sinyal. | SV_INT:priority = Prioritas. <br><br> SV_UINT:ratehz = Tingkat modulasi, dalam Hertz.<br><br> SV_UINT:waveshape = Bentuk gelombang modulasi. |
| SvEffectCreateI3dl2reverb | [Read Here](https://docs.microsoft.com/en-us/previous-versions/windows/desktop/ee417547(v=vs.85)) | SV_INT:priority = Prioritas. <br><br> SV_INT:room = Redaman efek ruangan, dalam milibel (mB).<br><br> SV_INT:roomhf = Redaman efek frekuensi tinggi ruangan, dalam mB.<br><br> SV_FLOAT:roomrollofffactor = Faktor rolloff untuk sinyal yang dipantulkan.<br><br> SV_FLOAT:decaytime = Waktu peluruhan, dalam detik.<br><br> SV_FLOAT:decayhfratio = Rasio waktu peluruhan pada frekuensi tinggi dengan waktu peluruhan pada frekuensi rendah.<br><br> SV_INT:reflections = Atenuasi refleksi awal relatif terhadap lRoom, dalam mB.<br><br> SV_FLOAT:reflectionsdelay = Waktu tunda refleksi pertama relatif terhadap jalur langsung, dalam detik.<br><br> SV_INT:reverb = Atenuasi gema akhir relatif terhadap lRoom, dalam mB.<br><br> SV_FLOAT:reverbdelay = Batas waktu antara pemantulan awal dan gaung akhir relatif terhadap waktu pemantulan pertama.<br><br> SV_FLOAT:diffusion = Kepadatan gema dalam peluruhan gema akhir, dalam persen.<br><br> SV_FLOAT:density = Kepadatan modal dalam peluruhan gema akhir, dalam persen.<br><br> SV_FLOAT:hfreference = Referensi frekuensi tinggi, dalam hertz |
| SvEffectCreateParameq | Equalizer parametrik memperkuat atau melemahkan sinyal dari frekuensi tertentu. | SV_INT:priority = Prioritas. <br><br> SV_FLOAT:center = Frekuensi tengah, dalam hertz.<br><br> SV_FLOAT:bandwidth = Bandwidth, dalam semitone.<br><br> SV_FLOAT:gain = Jumlah perubahan sinyal.|
| SvEffectCreateReverb | Equalizer parametrik memperkuat atau melemahkan sinyal dari frekuensi tertentu. | SV_INT:priority = Prioritas. <br><br> SV_FLOAT:ingain = Penguatan sinyal input, dalam desibel (dB).<br><br> SV_FLOAT:reverbmix = Campuran gema, dalam dB.<br><br> SV_FLOAT:reverbtime = Waktu gema, dalam milidetik.<br><br> SV_FLOAT:highfreqrtratio = Rasio waktu reverb frekuensi tinggi|

# Limitation

| Fungsi | Parameter |
| --     | --        |
| SvEffectCreateChorus | priority = 1 - 65535 <br> wetdrymix = 0.0 - 100.0 <br> depth = 0.0 - 100.0 <br> feedback = -99.0 - 99.0 <br><br> frequency = 0.0 - 10.0 <br><br> waveform = 0 - 1 <br><br> delay = 0.0 - 20.0 <br><br> phase = 0 - 4 ( 0, 1, 2, 3, 4)|
